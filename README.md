# Remove-Duplicates-from-Sorted-List

Given the head of a sorted linked list, delete all duplicates such that each element appears only once. Return the linked list sorted as well.

 

Example 1:
<img width="302" height="242" alt="image" src="https://github.com/user-attachments/assets/066d4c63-1b66-4f21-ac2a-44ac23cb1eb5" />


Input: head = [1,1,2]
Output: [1,2]
Example 2:
<img width="542" height="222" alt="image" src="https://github.com/user-attachments/assets/a549b6fb-e1e8-4531-ac6b-8d90c2dc837f" />


Input: head = [1,1,2,3,3]
Output: [1,2,3]
 

Constraints:

The number of nodes in the list is in the range [0, 300].
-100 <= Node.val <= 100
The list is guaranteed to be sorted in ascending order.
